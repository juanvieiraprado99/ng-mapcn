Tooltips Map Example

Mapa demonstrando tooltips customizados em markers. Passe o mouse sobre os markers para ver os tooltips.

Recursos demonstrados:

  • Tooltip habilitado/desabilitado com a propriedade enabled
  • Diferentes âncoras: top, bottom, left, right, top-left, bottom-right
  • Offset customizado (número ou array [x, y])
  • Tooltip que aparece no hover (showOnHover: true)
  • Marker sem tooltip
